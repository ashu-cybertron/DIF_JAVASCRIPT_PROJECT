Formatting Tags

  • b:bold <b>Hello</b> Output: Hello

  • strong:bold(for showing importance) <strong>Hello</strong> Output: Hello

  • u:underline <u>Hello</u> Output: Hello

  • i:italics <i>Hello</i> Output: Hello

  • em:italics(but used for emphasizing the text) <em>Hello</em> Output: Hello

  • sub:subscript CO<sub>2</sub> Output: CO2

  • sup:superscript x<sup>2</sup> Output: x2

Anchor Tag

Tag:<a></a>

Basically, anchor is used to link:

  1. To link to different sites
  2. To link to different page of same site
  3. To link to different sections of the same page

Color of the link

  • Blue: when not visited.
  • Purple: when already visited.
  • Red: when clicked.

To link to different sites: <a href="https://www.google.com">Click to Open Google</a>

Click to Open Google

To link to different page of same site: <a href="01.html">Click to open 1st page</a>

Click to open 1st page

To link to different section of same page: <a href="# followed by id of the element( hello: id of h4)">Hello h4 :)</a>

List Tags

There are mainly two types of lists:

  • Unordered list
  • Ordered list

Unordered List


ul: undordered list <ul> </ul>
li: list items <li> </li>

<ul>
<li> First Item </li>
<li> Second Item </li>
<li> Third Item </li>
</ul>
  • First Item
  • Second Item
  • Third Item

Attribute: <ul type="disc(default) | circle(hollow circle) | square" > </ul>

Ordered List


ol: ordered list <ol> </ol>
li: list items <li> </li>

<ol>
<li> First Item </li>
<li> Second Item </li>
<li> Third Item </li>
</ol>
  1. First Item
  2. Second Item
  3. Third Item

Attribute: <ol type="1(default) | i | I | A | a" > </ol>

ul>li*3 : it will create ul element with 3 list items

<ul>
   <li></li>
   <li></li>
   <li></li>
</ul>
